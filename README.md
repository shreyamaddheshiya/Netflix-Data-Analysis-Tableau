# 📊 Netflix Data Analysis Dashboard (Tableau)

## 🔍 Overview

This project analyzes Netflix data to identify trends in content distribution, genre popularity, and growth over time. The dashboard is built using Tableau for interactive visualization.

## 🎯 Objectives

* Analyze Movies vs TV Shows distribution
* Identify popular genres
* Study year-wise content growth
* Explore country-wise content

## 🛠️ Tools Used

* Tableau (Data Visualization)
* Microsoft Excel (Data Cleaning)
* CSV Dataset

## 📂 Dataset

The dataset contains information about Netflix titles including:

* Title, Type, Director
* Country, Release Year
* Rating, Duration, Genre

## 📈 Key Insights

* Movies dominate Netflix content
* Rapid growth after 2015
* Drama and Comedy are most popular
* USA contributes the highest content
